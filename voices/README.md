# Polish voices (XTTS-v2 speaker references)

The Polish engine (XTTS-v2) clones a voice from a short reference clip. Any
`*.wav` file you drop in **this directory** shows up as a selectable Polish
voice in the app (the voice name is the filename).

## How to add a voice

1. Record or find a **6–20 second** clip of clean Polish speech:
   - one speaker, no music or background noise
   - mono or stereo, ideally 24 kHz or higher
2. Save it here as e.g. `maria.wav`.
3. Refresh the app — `maria.wav` appears in the Polish voice dropdown.

## No voices yet?

If this folder has no `*.wav` files, Polish still works using the built-in
XTTS studio speaker **“Ana Florence”** (the app shows this as the only Polish
voice). Drop in a WAV to clone your own voice instead.

> Note: `*.wav` files here are git-ignored, so your voice samples stay local.
